The basic cammond in git....
1. mkdir directoryname
2. git init
3. touch filename
4. git status
5. git config
6. git config --global --list
7. git config --local --list
8. git config --global user.name username
9. git config --global --user.email emailid
10. git add filename
11. git add file*.txt
12. git add .
13. git commit -m "message"
14. git checkout filename
15. git checkout .
